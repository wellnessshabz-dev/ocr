# Ecosystem

The OpenCode ecosystem of plugins, integrations, and community tools.

## Official Plugins

- `@opencode/plugin-eslint` — ESLint integration
- `@opencode/plugin-prettier` — Prettier formatting
- `@opencode/plugin-jest` — Jest test runner

## Community Plugins

Browse npm for `@opencode/plugin-*` packages.

## Integrations

- GitHub (PRs, issues, CI) — built-in
- GitLab (MRs, issues, CI) — built-in
- Obsidian vault sync — external
- LLM Wiki — external

## Desktop App

Electron-based desktop app with TUI embed. Available for macOS, Windows, Linux.

## Web UI

Full-featured web interface for remote use.

## Share Server

Self-hosted conversation sharing server.

## SDK

Build your own integrations. See `docs_sdk.md`.
