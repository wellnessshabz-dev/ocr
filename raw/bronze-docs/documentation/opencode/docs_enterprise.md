# Enterprise

OpenCode Enterprise keeps code and data on your infrastructure via centralized config, SSO, and internal AI gateway.

## Data Handling

OpenCode does not store your code or context data. All processing happens locally or through direct API calls. The `/share` feature is the only exception.

## Code Ownership

You own all code produced by OpenCode. No licensing restrictions.

## Deployment

- **Central Config** — Single config for entire org
- **SSO Integration** — Via central config with identity management
- **Internal AI Gateway** — Route all requests through your infrastructure
- **Self-hosting** — Share pages self-hosting (on roadmap)

## Private NPM Registry

OpenCode supports private npm registries through Bun's `.npmrc` support.

## Pricing

Per-seat model. No token charges if using your own LLM gateway.
