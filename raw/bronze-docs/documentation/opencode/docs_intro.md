# Intro

Get started with OpenCode — an open source AI coding agent available as a terminal-based interface, desktop app, or IDE extension.

## Prerequisites

- A modern terminal emulator (WezTerm, Alacritty, Ghostty, Kitty)
- API keys for LLM providers

## Installation

```bash
curl -fsSL https://opencode.ai/install | bash
```

Alternative methods: npm (`npm install -g opencode-ai`), Homebrew (`brew install anomalyco/tap/opencode`), Arch Linux, Windows (Chocolatey, Scoop, NPM), Docker (`ghcr.io/anomalyco/opencode`).

## Quick Start

1. Run `/connect` in the TUI to configure a provider
2. Navigate to a project and run `opencode`
3. Run `/init` to create an AGENTS.md
4. Start coding!

## Features

- Plan mode for analysis without changes (switch via Tab)
- Undo/redo changes (`/undo`, `/redo`)
- Share conversations (`/share`)
- Image attachment via drag-and-drop
