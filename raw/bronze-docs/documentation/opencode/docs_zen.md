# Zen Mode

Zen is a full-screen, distraction-free coding mode in the TUI.

## Activation

- Press `z` in any mode
- Or use `/zen`
- Exits with `z` or `/zen` again

## Features

- Hides all panels except the conversation
- Minimal UI chrome
- Keyboard shortcuts still active
- Auto-hides status bar

## Use Cases

- Deep focus sessions
- Code review with minimal distraction
- Presentation / demo mode
